The table below highlights the 30 Democrats in the 116th House representing districts won by DJT in 2016. Full data set available here. Methods for building data set available here.

Updated per Jeff Van Drew’s party switch.

library(tidyverse)
url1 <- 'https://raw.githubusercontent.com/jaytimm/uspoliticalextras/master/clean-data-sets/thirty-one-house-democrats.csv'
fl30 <- read.csv(url(url1)) %>% 
  rename(dist = district_code, 
         st = state_abbrev, 
         rep = house_rep,
         rep_margin = house_rep_margin, 
         fr = is_fresh) %>%
  select(-bioguide_id, -house_flip)

fl30 %>%
  knitr::kable()
